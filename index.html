EJERCICIO GRÁFICOS

Isabela Aylwin & Natalia Valera

Para el desarrollo de este ejercicio se utilizaron distintas plataformas para rescatar referentes que encajaran en las clasificaciones de los tipo de gráficos definidos en la clase. Para los Gráficos Figurativos y los Gráficos Mixtos se ejemplificó mediante imágenes en formato .jpg, mientras que para el Gráfico no Figurativo se utilizó un recurso .svg resultando en un gráfico de barras.

GRÁFICA FIGURATIVA

Los gráficos figurativos se caracterizan por representar fenómenos físicos, en lo que esta presente la forma visual en la representación.

GRÁFICA MIXTA

Los gráficos mixtos contienen características tanto de gráficos figurativos como los gráficos no figurativos.

GRÁFICA NO FIGURATIVA

Los gráficos no figurativos muestran de forma abstracta lo que sería difícil de representar a través y es capaz de visualizar la cantidad estadística. Para la construcción de este gráfico se utilizaron datos y herramientas obtenidos desde Rawgraphs 2.0 Beta. Se escogió la base de datos de las series originales lanzadas por la plataforma de streaming Netflix entre 2013 y 2017, y se las clasificó segun género y su calificación obtenida en IMDB.

La gráfica de arriba se despliega en la página como si se tratara de una imagen rasterizada, con el elemento <img>. Otros elementos de HTML podrían usarse para un mejor despliegue del SVG. Reviseramos tales elementos algunas clases más adelante, pero ya les conviene revisar Using SVG | CSS Trick.
